# Procédure : Réinitialisation d’un mot de passe AD

## Étapes
1. Rechercher l’utilisateur dans ADUC  
2. Clic droit → **Reset Password**  
3. Entrer le nouveau mot de passe temporaire  
4. Cocher **User must change password at next logon**  
5. Valider  

## Vérifications
- Aucun message d’erreur  
- L’utilisateur peut se reconnecter
