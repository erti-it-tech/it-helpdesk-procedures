# Procédure : Configuration d’un compte Microsoft 365 sur un nouveau PC

## Étapes
1. Ouvrir Outlook  
2. Ajouter un compte → Email M365  
3. Authentification via Azure AD  
4. Validation MFA (si activée)  
5. Synchronisation du profil  
6. Vérifier Teams  
7. Vérifier OneDrive (le nuage doit être bleu)  

## Vérifications
- Boîte mail synchronisée  
- Teams connecté  
- OneDrive synchronisé
