# Procédure : Préparation d’un nouveau PC (Workplace)

## Étapes
1. Mise à jour Windows  
2. Installation Office Microsoft 365  
3. Installer Teams  
4. Activer OneDrive  
5. Installer navigateur (Edge/Chrome)  
6. Installer imprimantes  
7. Vérifier :
   - AD login OK  
   - Outlook OK  
   - OneDrive OK  

## Vérifications finales
- PC prêt pour l’utilisateur  
- Aucun message d’erreur  
- Performance OK
