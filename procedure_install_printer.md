# Procédure : Installation d’une imprimante

## Étapes
1. Windows + R → `control printers`  
2. **Add a printer**  
3. Choisir :  
   - imprimante réseau  
   - ou installation via pilote  
4. Si imprimante réseau :
   - Entrer `\\SERVEUR-PRINT\NomImprimante`  
5. Installer le pilote automatiquement  

## Vérifications
- Page de test OK  
- L’imprimante apparaît dans les périphériques
