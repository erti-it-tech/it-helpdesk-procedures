# Procédure : Déverrouillage d’un compte utilisateur

## Symptôme
L'utilisateur a trop tenté de se connecter → compte verrouillé.

## Étapes
1. Ouvrir ADUC  
2. Propriétés de l’utilisateur  
3. Onglet **Account**  
4. Décocher **Account is locked out**  
5. Valider  

## Vérification
- L’utilisateur peut se reconnecter
